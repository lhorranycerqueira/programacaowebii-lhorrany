- border - border - border: 1px solid;
- cellpadding - padding - padding: 10px;
- cellspacing - border-spacing - border-spacing: 5px;

### vh e vw
um vw viewport width - 1vw equivale a 1% da largura da tela
um vh viewport height - 1vh equivale a 1% da altura da tela
tamanho da tela eh mais importante !
