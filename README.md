## Anotações da aula 17/03
atributos pra mudar cor, tamanho ou alinhamento e foram substituidos pelo css
nao se usa mais no html !
html - css
align - text-align/float - text-align: center;
bgcolor - background-color - background-color: #f0f0f0
border - border - border: 1px solid;
cellpadding - padding - padding: 10px;
cellspacing - border-spacing - border-spacing: 5px;
width e height- witdh/height - width: 100%
text - color - color:blue
vlink, alink, link, text - pseudo-classes: :link, :visited, :active; 
clear - clear - clear: both;
## pseudo-classes
estado especial, movimento, quando passa o mouse por cima do link e a cor muda.

1rem ou 1em - tamanho original
2rem ou 2em - por 2x
0.5rem ou 0.5em - diminuir por 0.5x
